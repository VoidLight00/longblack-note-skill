## 본문 추출 규칙

브라우저 evaluate로 `.note-content` 내 요소 순회:

| HTML 요소 | 마크다운 변환 |
|-----------|-------------|
| `<h3>` | `## 🔥 Chapter N. 제목` |
| `<h4>` | `### 소제목` |
| `<p>` | 본문 텍스트 (strong/em 보존) |
| `<blockquote>` | `> **"텍스트"**` (핵심은 `> **=="텍스트"==**`) |
| `<figure>` | 이미지 제외 |
| `<figcaption>` | `> [!info]- 사진 설명` |

## 파일명 규칙
`YYYY-MM-DD-제목-요약.md` (한글, 하이픈 구분, 20자 이내)
